Proyecto

Plataforma web que nos permite obtener información sobre jugadores de la plataforma Steam. Nos permitirá organizar de forma sencilla distintos grupos de personas para formar partidas (Calendario de quedadas, organización de clanes).

Utilidades
  • Sistema que nos permitirá organizar y crear clanes. Cualquier usuario de la plataforma podrá crear un clan, así el mismo sea el administrador de ese clan. El administrador de la web solo se encargará de asegurarse de que los clanes no tengan un nombre ofensivo, la administración interna del clan será organizada por el creador y los miembros del clan.

  • Sistema para crear partidas entre clanes, se podrá enviar peticiones de batallas entre distintos clanes. El clan retado decidirá si quiere aceptar el desafío y ambos clanes una vez aceptado el reto deberán decidir a sus participantes.

  • Sistema para crear partidas entre clanes que dispongan de un mismo nivel. El sistema creará un emparejamiento entre dos clanes que tengan un ELO parecido.

  • Sistema para crear partidas conjuntas entre distintos jugadores, la página dispondrá de un calendario conjunto donde el usuario podrá ver todos los eventos que estén disponibles y tendrá la opción de unirse a uno. Todos los eventos son creados por usuarios, por lo que todos los usuarios tienen la capacidad de crear eventos además de unirse a eventos ya creados.

  • Sistema para crear partidas entre usuarios que dispongan de un mismo ELO. El sistema emparejará a los usuarios con un mismo ratio de victorias que otros jugadores.
Usuarios

La plataforma va dirigida a todos esos usuarios que quieran jugar con otras personas que no solo sean sus amigos. Se le permitirá al usuario conseguir partidas con otros usuarios que busquen partidas con los mismos juegos que el usuario.

También va destinado a los usuarios que quieran organizar un poco se forma de jugar. El sistema le permite formar parte de distintos clanes con los que jugar con gente, obtener información…

Diferencia con otras plataformas

Nuestra plataforma se diferenciaría del resto de sistemas en la creación de clanes de batalla, esta funcionalidad permite a los usuarios realizar partidas y formar grupos para jugar de una forma mucho más sencillo.
